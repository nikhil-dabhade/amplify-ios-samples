## Amplify iOS Predictions

### Setup
**Clone repo**
```bash
$ git clone 
$ cd 
```

**Install dependencies**
- Amplify CLI 
https://aws-amplify.github.io/docs/cli-toolchain/quickstart#installation

```bash
$ npm install -g @aws-amplify/cli
```
